# Landing Page Course Template

## Descrizione
Template modulare per landing page di corsi online con integrazione pagamenti e lead generation.

## Componenti
- `index.html`: Struttura principale
- `styles.css`: Stili globali
- `components/`: Cartella con componenti modulari
- `scripts/`: Logica interattiva

## Requisiti
- HTML5
- CSS3
- Vanilla JavaScript
- Integrazione Stripe (opzionale)

## Installazione
1. Clona repository
2. Personalizza contenuti
3. Configura integrazioni

## Configurazione
Modificare `config.js` per personalizzare:
- Dettagli corso
- Prezzi
- Impostazioni form
